argon2_dart

argon2_dart library Null safety

Library for Argon2 Hashing and verification in dart prrojects

To use this library in your code:

import 'package:dargon2/dartgon2.dart';

For detailed usage please look at the example

Classes

A class that houses all of the methods to hash and verify a password using the Argon2 password hashing algorithm. [...]

DArgon2Result
A class that stores the results from an Argon2 Hashing operation. Allows for a user to get various values from the List _bytes and _encodedBytes.
Salt
A class to abstract and handle Salts for the Argon2 hashing process. Allows for a user to create a salt with given bytes (List) or create a new salt with secure random bytes.

DArgon2ErrorCode
The enum with all of the Argon2 Error codes, seen directly from the C Reference Library of Argon2. The codes are sent into the Exception object as well as used to retrieve the error message from the C library.
